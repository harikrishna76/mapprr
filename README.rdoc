Instructions:

1. Clone git repository https://github.com/harikrishna76/mapprr.git 
2. Setup mongodb, and make sure you are giving exact host link of mongodb(ex:'mongodb://127.0.0.1:27017') in the config/mongodb.yml
3. Install the bundler
4. Run the rake task(rake db:seed) to seed the dummy data
5. Now you can start the server
